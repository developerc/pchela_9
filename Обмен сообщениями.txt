С компьютера диспетчера отсылается серверу HTTP запрос GET
http://pchelka.teleknot.ru/api/lgn/psw/reforders
В ответ клиент-сокет сервера HTTP отсылает сервер-сокету компьтера
диспетчера на порт: 
1 компьютер 6004
2 компьютер 6005
текст REF
При получении сервер-сокетом этого текста программа обновляет таблицы

клиент отсылает серверу HTTP запрос GET
http://pchelka.teleknot.ru/api/lgn/psw/queue
в ответ получает страничку с таблицей очереди машин

клиент отсылает серверу HTTP запрос POST
http://pchelka.teleknot.ru/api/lgn/psw/order
с параметром 
uvedomlen равным 2, 3 или 4
сервер HTTP записывает в поле uvedomlen таблицы zakazi это значение
для отображения уведомления водителя о получении заказа

При запуске программы программа регистрируется на сервере
клиент отсылает серверу HTTP запрос POST 
http://pchelka.teleknot.ru/api/lgn/psw/dispreg
без параметров


клиент отсылает серверу HTTP запрос POST 
http://pchelka.teleknot.ru/api/lgn/psw/qpush
с параметром 
point_id= равным от 0 до 10  это номер стоянки
state= равным 0 если отправляет диспетчер, 4 если отправляет андроид водителя
для перестановки на новую точку

клиент отсылает серверу HTTP запрос POST 
http://pchelka.teleknot.ru/api/lgn/psw/qpush
с параметром 
row= равным от 0 до 10  это номер стоянки
state= равным 0 если отправляет диспетчер, 4 если отправляет андроид водителя
delzak=1 удалять из таблицы заказов (delzak=0 не удалять) 
для перестановки на новую точку

клиент отсылает серверу HTTP запрос POST 
http://pchelka.teleknot.ru/api/lgn/psw/orderdestroy
без параметров
для удаления строки из таблицы zakazi
и обновления(добавления) в таблицу zvonki
если response не содержит "none" то в таблице zakazi больше одной строки с таким водителем

При нажатии Enter в таблице заказов в поле "машина" клиент отсылает серверу HTTP запрос POST 
на постановку машины на заказ
http://pchelka.teleknot.ru/api/lgn/psw/caronorder
c параметрами:
car: машина
order_id: номер заказа
  
При отработке заказа данные из строки заказа переносятся в таблицу zvonki, отсылаем POST запрос
http://pchelka.teleknot.ru/api/lgn/psw/zvonkiupd
с параметром:
car_id: номер машины для которой отрабатывается заказ

Смена состояния машины без перестановки в очереди (подсвечивание синим или красным ячейки)
http://pchelka.teleknot.ru/api/lgn/psw/zvonkiupd
с параметром:
state: состояние машины
car: машина (для водителя не обязательно)
